# HeyGen Pipeline Changelog

## 1.0.0 (2024-02-10)

### Initial Release

- **Core Features:**
  - HeyGen API v2 integration
  - Script-to-video generation
  - Multi-language support
  - Client/campaign organization
  - Batch processing with queue management
  - Video storage and retrieval
  - CLI interface

- **Commands:**
  - `create` - Create single video
  - `batch` - Process multiple videos
  - `list` - List all videos
  - `status` - Check video/job status
  - `download` - Download completed video
  - `avatars` - List available avatars
  - `voices` - List available voices
  - `report` - Generate storage report
  - `cleanup` - Clean up old videos

- **Examples:**
  - Multi-language campaign
  - Client ad campaign
  - Internal training series
  - Social media content

- **Templates:**
  - Product demo
  - Social media hook
  - Testimonial
  - Announcement
